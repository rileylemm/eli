Creating Icons for Reddit Explainer

This extension requires three icon sizes: 16x16, 48x48, and 128x128 pixels. The placeholder files have been created, but you'll need to replace them with actual icon images.

Icon Requirements

  • File format: PNG with transparency
  • Sizes needed: 16x16, 48x48, and 128x128 pixels
  • Style: Simple, recognizable at small sizes
  • Suggested theme: Use Reddit's orange (#FF5700) as the primary color

Suggested Icon Design

A simple design could be a speech bubble in Reddit orange with a simplified "ELI5" text inside, or perhaps a combination of the Reddit alien logo with a lightbulb or speech bubble.

How to Create Icons

  1. Use a graphic design tool like Adobe Illustrator, Sketch, or a free alternative like GIMP or Inkscape
  2. Create your icon at the largest size (128x128) first
  3. Export smaller versions, making sure to optimize them for smaller sizes (simplify details as needed)
  4. Save all files as PNG with transparency
  5. Replace the placeholder files in the images directory

Placeholder Icons

For quick testing, you can use solid color squares with "R16", "R48", and "R128" text on them, just to have something visible while testing.

Online Icon Generators

If you don't have design software, you can use an online icon generator or favicon generator. Many of these can create icons in multiple sizes from a single image.

After creating your icons, replace the files in the images directory and delete this instructions file.
